# Project Interview

## Round 1: Vision
Question: 이 메모앱은 어떤 플랫폼에서 동작해야 하나요?
Answer: 모든 플랫폼 (모바일 + 데스크톱 + 웹) — Flutter로 iOS, Android, macOS, Windows, Linux, Web 전체 지원

## Round 2: Technology
Question: 음성 기능의 범위를 선택해주세요.
Answer: 음성 녹음 → STT 텍스트 변환 — 음성을 녹음하면 자동으로 텍스트로 변환하여 메모에 저장. Whisper API 또는 디바이스 네이티브 STT 활용

## Round 3: Scope
Question: 메모앱의 핵심 기능과 범위를 선택해주세요.
Answer: 태그/폴더 + 검색 + 음성 STT + 동기화 — 메모 조직화(태그/폴더), 풍부한 검색, 음성, 동기화 비롯하기. 생산성 도구에 가까운 범위.
